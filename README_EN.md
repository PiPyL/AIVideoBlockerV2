# SafeKid V2 - Protect Kids from AI-Generated Content on YouTube

SafeKid is an open-source Chrome Extension (Manifest V3) that supports blocking channels or videos unsuitable for children, either configured manually by parents or through an automatic detection mechanism.

[Tiếng Việt](./README.md) | [English](./README_EN.md)

## ✨ Key Features

- **Multi-layer Detection:** Combines YouTube native labels, NLP heuristics, and keyword databases.
- **Gemini AI Integration:** Utilizes the Gemini 3.1 Flash model to intelligently analyze video context and thumbnails.
- **Parental Security:** Settings are protected by a secure parental password.
- **Privacy First:** Operates entirely within your browser; no user data is collected.

## 🚀 Installation (Manual)

The extension is currently pending approval on the Chrome Web Store. You can install it manually:

1. Download [safekid-extension.zip](https://github.com/PiPyL/safekid-extension/raw/main/safekid-extension.zip) and extract it.
2. Navigate to `chrome://extensions/` in your Chrome browser.
3. Enable **Developer mode** (top right corner).
4. Click **Load unpacked** and select the folder you just extracted.
